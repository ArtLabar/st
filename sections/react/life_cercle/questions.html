Что делает метод жизненного цикла componentDidMount()

Этот метод вызывается автоматически сразу после монтирования компонента в DOM.

Когда вызывается метод жизненного цикла componentDidUpdate(prevProps, prevState, snapshot)?

Метод вызывается сразу после обновления компонента. При первом монтировании не вызывается.

Когда вызывается метод жизненного цикла componentWillUnmount(prevProps, prevState, snapshot)?

Метод вызывается перед размонтированием и удалением компонента.

Когда вызывается componentDidCatch(err, info)?

Метод вызывается когда в компоненте или любом из его предков возникла ошибка. err - это перехваченная ошибка, info, объект, где в свойстве componentStack содержится информация о компоненте, в котором возникла ошибка.

Для чего нужны рефы?

Они позволяют получить доступ к DOM-узлам и React-элементам, созданным в рендер-методе.

Что делает React.createRef()?

Создает объект реф. Он должен быть присвоен атрибуту ref в нужном DOM-элементе или React-компоненте.

                    
class MyComponent extends React.Component {
    constructor(props) {
        super(props);
        this.myRef = React.createRef();
    }

    render() {
        return 
;
    }
}

Через какое свойство рефа можно получить нужный узел?

Когда реф передаётся элементу в методе render, ссылка на данный узел доступна через свойство рефа current.

                    
const node = this.myRef.current;

Почему атрибут ref можно использовать в классовых компонентах и нельзя в функциональных?

Когда ref используется с классовыми компонентами, свойство current получает экземпляр смонтированного компонента. А для функциональных компонентов экземпляр не создается.

В какой момент происходит обновление свойства current рефа?

Обновление происходит перед вызовом методов componentDidMount() и componentDidUpdate().

Как можно вызвать метод дочернего компонента, имя на него ссылку через реф?

Поскольку компонент это объект то метод его можно вызвать так же, как и в любом другом объекте: obj.method()

                    
class AutoFocusTextInput extends React.Component {
    constructor(props) {
        super(props);
        this.textInput = React.createRef();
    }
    
    componentDidMount() {
        this.textInput.current.focusTextInput();
    }
    
    render() {
        return (
        
        );
    }
}
