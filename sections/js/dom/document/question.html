Как можно получить доступ к элементу document, html, body?

                    
document;

document.documentElement;

document.body;

Перечислите свойства навагации элемента, учитывая навигацию по текстовым узлам. 

                    
parentNode;
previousSibling;     nextSibling;
childNodes;
firstChild;       lastChild;

Является ли список элементов, полученный при помощи querySelectorAll() “живым” списком?

Да.

Какие существуют свойства у элемента для навигации только по узлам-элементам?

                    
parentElement;
previousElementSibling; nextElementSibling;
children;
firstElementChildren; lastElementChildren;

Существуют ли дополнительные свойства у таблицы для навигации?

Да.

Можно ли вызвать метод getElementById() на любом элементе?

Нет. Это метод узла document.

Почему нельзя использовать переменную, соответствующую id элемента для поиска этого элемента.

Это плохая практика, к тому же, данная переменная может быть перезатерта из глобального лексического окружения.

Можно ли вызвать метод querySelectorAll() на любом элементе?

Да.

Чем отличается querySelectorAll() от querySelector()?

Первый возвращает коллекцию элементов, второй метод - первый элемент из коллекции.

Что делает метод элемента marches()?

Проверяет, соответствует ли элемент определенному селектору CSS. Возвращает true или false.

Что делает метод элемента closest()?

Проверяет, соответствует ли элемент указанному селектору. Если нет, то ищет совпадение в элементах-продках вверх по цепочке. Возвращает первый соответствующий элемент.

Что показывает свойство узла nodeType?

Номер, который соответствует определенному типу данного узла. Например: 1 - элемент, 3 - текстовый узел, 9 - узел document и т.д.

Что показывают свойства tagName и nodeName? В чем разница между ними?

Показывают тип узла. Свойство tagName есть только у элементов и покажет имя тега в верхнем регистре. Cвойство nodeName для элемента покажет имя элемента, для текстового узла - строку текста.

Что делает свойство innerHTML?

Оно позволяет либо получить содержимое элемента, либо перезаписать его.

Что делает свойство outerHTML?

Оно позволяет получить разметку элемента целиком (включая внешние теги) либо заменить его.

Что делают свойства текстовых узлов nodeValue и data и в чем разница между ними?

Они почти идентичны в использовании, разница только лишь в спецификации. Они позволяют получить текстовое содержимое текстового узла. Это аналог innerHTML, только для текстовых узлов.

Что делает свойство textContent? Какое значение ему можно присвоить?

Позволяет получить текстовое содержимое элемента без тегов. Если ему присвоить строку, в которой будут содержаться теги, то js экранирует их и вставить в элемент как строки.

Что показывает свойство элемента hidden?

Оно показывает, виден или нет элемент в данный момент. Можно присвоить логические значения true или false.

Являются ли атрибуты html-элемента свойствами соответствующего объекта Element?

Да. Но это справедливо только для стандартных атрибутов, установленных спецификацией. Эти свойства служат как для получения значений, так и для записи.

                    
elemImage.alt='Изображение';
elemImage.src='https://github.com';
elemInage.title='Изображение';

Расскажите, что делают следующие методы элементов и какие аргументы они принимают: .hasAttribute(), .getAttribute(), setAttribute(), removeAttribute()?

Проверяет, получает, устанавливает и удаляет атрибут соответственно. Эти методы позволяют работать с нестандартными свойствами в том числе.

Что позволяет получить свойство элемента attributes?

Он позволяет получить итерируемый объект объектов, каждый из которых имеет свойства name и value (каждый объект соответствует одному атрибуту элемента).

Если изменить свойство объекта Element, изменится ли атрибут?

Да.

Если изменить атрибут, изменится ли свойство объекта Element?

Да. Изменения происходят синхронно.

Можно ли изменить свойство value объекта Element, меняя атрибут? А изменится ли атрибут, если изменить свойство value?

Нет. Они не взаимосвязаны. Атрибут value ссылается на свойство defaultValue, а свойство value на фактические данные, указанные в соответствующем поле формы. 

                    
let elem = document.createElement('input');
elem.setAttribute('value', 'Hello');
elem.value; // 'Hello';
elem.value = 'Good bay';
elem.getAttribute('value'); // "Hello";
elem.setAttribute('value', 'Hi');
elem.value; // 'Good bay';

Что мы получим, если захотим получить значение атрибута style как свойство узла и вызвав метод getAttribute()?

Через свойство получим объект, через метод получим строку со стилями.

Какие значения можно получить при elem.href и elem.getAttribute(‘href’)?

В первом случае получить полный url вместе с http-протоколом, во втором значение атрибута, как записано в документе.

Согласно спецификации, любому элементу можно назначит нестандартный атрибут, написание которого начинается с приставки data-. Как можно получить значения таких атрибутов?

Объект Element имеет свойство dataset, которое ссылается на объект со всеми атрибутами data-. Каждый атрибут соответствует свойству из этого объекта, но без приставки -data. Если в атрибуте есть тире, то используется CamelCase.

                    


div.dataset.nameAinmal; // 'bars'
div.dataset.namePets = 'Lola';
div; //

Также, можно пользоваться методами для работы с атрибутами: getAttribute/setAttribute и так далее.

                    
div.getAttribute('data-name-pets'); // 'Lola'

Как можно создать новый DOM-элемент?

                    
document.createElement('div');

В аргументе принимает имя нужного тега.

Как можно создать новый текстовый узел?

                    
document.createTextNode("Hello, world");

Перечислите методы элемента документа, при помощи которых можно вставить новый DOM-элемент или текстовый узел.

Методы вызываются на DOM-элементе, относительно которого производится вставка.

                    
elem.append();
elem.prepend();
elem.after();
elem.before();
elem.replaceWith();

elem - это узел который уже существует в документе. Методы принимают строку либо узел.

Что делают методы элемента insertAdjacentHTML(), insertAdjacentText(), insertAdjacentElement().

Данные элементы позволяют вставить строку с html-разметкой, текст или элемент в определенное место в документе относительно данного элемента. 

                    
elem.insertAdjacentElement('afterbegin', newElement);

Первым аргументом принимается флаг, показывающий в какое место нужно вставить узел. Есть 4 вида флага: beforebegin, afterbegin, beforeend, afterend - они покузывают перед или после открывающего или закрывающего тега нужно вставить узел. Вторым аргументом передается вставляемый узел.

При помощи какого метода можно удалить элемент?

                    
elem.remove();

Можно ли взять элемент из одной части документа и перенести в другое?

Да, можно. Нужно выбрать элемент и вставить его в нужное место при помощи методов вставки. Со своего старого места элемент будет удален (элемент не дублируется).

При помощи какого метода можно сделать копию элемента? Какие аргументы принимает данный метод?

                    
elem.cloneNode(true)

Метод принимает значение булево типа. Если передаем true, будет создана глубокая копия, если false - поверхностная без вложенных элементов.

Что делает конструктор new DocumentFragment()?

Он создает узел, который выступает в роли контейнера-обертки. В него можно добавлять дочерние элементы. Когда данный узел вставляется в документы, будут вставлены только дочерние элементы без обертки. 

                    
const wrapped = new DcoumentFragment();
wrapped.append(anotherElement);

На что ссылается свойство elem.style?

На объект CSSStyleDecloration, через который можно получить инлайновые стили либо установить стили инлайново.

Что можно получить при помощи свойства элемента className?

Строку с перечисленными через пробела классами элемента. Если присвоить данному свойству значение, то оно полностью перезатрет предыдущее.

                    
elem.className;  // 'main-class'
elem.className = "arrow arrow__left arrow__left--focus";
elem.className;  // "arrow arrow__left arrow__left--focus";

На что ссылается свойство элемента classList?

Оно ссылается на псевдомассив со всеми классами элемента. Это перебираемый объект и его можно обойти при помощи цикла for...of.

Перечислите методы для работы с классами в объекте classList?

                    
elem.classList.add("class");
elem.classList.remove("class");
elem.classList.toggle("class");
elem.classList.contains("class");

Что делает свойство elem.style.cssText?

С его помощью можно получить строку со всеми инлайновыми стилями. Также, с его помощью можно записать инлайновые стили, присвоив ему строку с нужными css-правилами. 

                    
div.style.cssText=`color: red !important;
    background-color: yellow;
    width: 100px;
    text-align: center;
`;

Это свойство редко используется, потому что такое присваивание удаляет все существующие стили: оно не добавляет, а заменяет их. Можно ненароком удалить что-то нужное. Но его можно использовать, к примеру, для новых элементов, когда мы точно знаем, что не удалим существующий стиль.

То же самое можно сделать установкой атрибута: div.setAttribute('style', 'color: red...').

Как можно убрать инлайновое свойство стиля, добавленное ранее?

Нужно свойству присвоить значение пустой строки.

                    
document.body.style.display = ""

Как можно получить окончательные стили элемента после применения к нему каскадности?

Данное свойство возвращает объект с вычесленными окончательынми стилями.

                    
elem.getComputedStyle(elem, pseudo);

pseudo указывается, если нужен стиль псевдоэлемента, например ::before. Пустая строка или отсутствие аргумента означают сам элемент.
