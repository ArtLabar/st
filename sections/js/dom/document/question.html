Как можно получить доступ к элементу document, html, body?

document, document.documentElement и document.body соответственно.

На что ссылаются такие свойства DOM-узла, как parentNode, previousSibling, nextSibling, childNodes, firstChild, lastChild?

На родительский элемент, предыдущего соседа, следующего соседа, псевдомассив с дочерними элементами, первый дочерний элемент и последний дочерний элемент соответственно. В данной системе навигации учитываются и текстовые узлы.

Являются ли DOM-коллекции живыми?

Да.

Какие существуют свойства элемента для навигации только по узлам-элементам?

parentElement, previousElementSibling, nextElementSibling, children, firstElementChildren, lastElementChildren.

Существуют ли дополнительные свойства у таблицы для навигации?

Да.

Можно ли вызвать метод getElementById() на любом элементе?

Нет. Это метод узла document.

Почему нельзя использовать переменную, соответствующую id элемента для поиска этого элемента.

Это плохая практика, к тому же, данная переменная может быть перезатерта из глобального лексического окружения.

Можно ли вызвать метод querySelectorAll() на любом элементе?

Да.

Чем отличается querySelectorAll() от querySelector()?

Первый возвращает коллекцию элементов, второй первый элемент из коллекции.

Что делает метод элемента marches()?

Проверяет, соответствует ли элемент определенному селектору CSS.

Что делает метод элемента closest()?

Находит первый элемент, от текущего и по цепочке предков, который соответствует селектору CSS.

Что показывает свойство узла nodeType?

Номер, который соответствует определенному типу узла. Например: 1 - элемент, 3 - текстовый узел, 9 - узел document.

Что показывают свойства tagName и nodeType? В чем разница между ними?

Показывают тип узла. Свойство tagName есть только у элементов и покажет имя тега в верхнем регистре.

Что делает свойство innerHTML?

Оно позволяет либо получить содержимое элемента, либо перезаписать его.

Что делает свойство outerHTML?

Оно позволяет получить разметку элемента целиком (включая внешние теги) либо заменить его.

Что делают свойства текстовых узлов nodeValue и data и в чем разница между ними?

Они почти идентичны в использовании, разница только лишь в спецификации. Они позволяют получить текстовое содержимое текстового узла. Это аналог innerHTML как у элементов.

Что делает свойство textContent? Какое значение ему можно присвоить?

Позволяет получить текстовое содержимое элемента без тегов. Если ему присвоить строку, в которой будут содержаться теги, то js экранирует их и вставить в элемент как строки.

Что показывает свойство элемента hidden?

Оно показывает, виден или нет элемент в данный момент. Можно присвоить логические значения true или false.

Становятся ли стандартные атрибуты html-элемента свойствами соответствующего объекта в DOM?

Да. Стандартные атрибуте это те, которые есть у html-элемента по спецификации.

Расскажите, что делают следующие методы элементов и какие аргументы они принимают: .hasAttribute(), .getAttribute(), setAttribute(), removeAttribute()?

Проверяет, получает, устанавливает и удаляет атрибут соответственно. Эти методы позволяют работать с нестандартными свойствами в том числе.

Что позволяет получить свойство элемента attributes?

Он позволяет получить коллекцию объектов, каждый из которых имеет свойства name и value (каждый объект соответствует одному атрибуту элемента).

Если изменить стандартный атрибут через свойство узла-элемента. Получить ли новое значение атрибута вызов getAttribute()?

Да. Изменения происходят синхронно.

Если мы меняем значение стандартного атрибута через setAttribute(), можно ли получить новое значение данного атрибута через свойство узла-элемента?

Да. Изменения происходят синхронно.

Можно ли изменить атрибут value изменяя свойство элемента? А изменится ли свойство, если изменить атрибут value?

Изменение атрибута обновляет свойство, но изменения свойства не изменяет атрибут.

Что мы получим, если захотим получить значение атрибута style как свойство узла и вызвав метод getAttribute()?

Через свойство получим объект, через метод получим строку со стилями.

Если нужно получить значение атрибута href. Что вернет DOM-свойство и что вернет вызов метода getAttribute()?

DOM-свойство вернет полный адрес url, метод именно то, что записано в html-документе.

Как получить доступ к нестандартному элементу, начинающиеся с приставки data-?

При помощи свойства элемента dataset, которое ссылается на объект со всеми такими атрибутами. Получить конкретный атрибут из данного объекта можно при помощи имени, которое указывается в самом атрибуте после приставки. Если в имени есть дефис, то используется верблюжья нотация.

Объясните, что такое "частичное применение"?

При помощи встроенного метода .bind() можно привязать не только this, но и аргументы.

Объясните, что такое "частичное применение"?

При помощи встроенного метода .bind() можно привязать не только this, но и аргументы.

            
function mul(a, b) {
return a * b;
}

let double = mul.bind(null, 2);

double(3) // = mul(2, 3) = 6
double(4) // = mul(2, 4) = 8
double(5) // = mul(2, 5) = 10

В качестве this можно передать любой объект, но как правило это null.
