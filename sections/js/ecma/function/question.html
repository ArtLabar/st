Какое значение будет присвоено аргументу функции, если при вызовы ему не присвоить значение?

В таком случает в теле функции аргументу будет присвоено значение undefind.

                    
function showArgument(x){
    return x;
}

showArgument() // undefind

Обратите внимание, что в вырожение может быть довольно сложным. Вы заметили висячую запятую?

Можно ли задать значение аргумента функции по умолчанию?

Да, это можно сделать в момент определения функции.

                    
function showArgument(x=10){
    return x;
}

showArgument() // 10
showArgument(40) // 40

Обратите внимание, что в вырожение может быть довольно сложным. Вы заметили висячую запятую?

Что вернет функция, если в ней не указан return или return указан пустой?

И в том и в другом случае функция вернет underfind.

                    
function showArgument(x=10){
    alert(x);
}

function getArgument(x){
    alert(x);
}

showArgument() // underfind;
getArgument(40) // underfind;

Обратите внимание, что в функции showArgument заданно значение по умолчанию.

Почему хорошо, когда одна функция выполняет одно действие?

Такие функции понятные и хорошо поддерживаются. Кроме этого, такие функции сами по себе являются отличными комментариями.

Объясните, как вы понимаете понятия Function Declaration и Function Expression?

Эти понятия обозначают разные способы определения функций. Функции, объявленные при помощи function declaration всплывают и видны в любом месте скрипта или блока кода. В строгом режиме такая функция за пределами блока кода будет не видна!!!

                    
//function declaration

function foo(){
    ...code
}

//function expression 

let foo = function(){
    ...code
};

Обратите внимание, что после во втором случаи стоит ";". После блоков кода и синтаксических конструкций(циклы, function declaration, условные конструкции) ставить точку с запятой. Однако во втором случае идет операция присваевания значения переменной и именно эту операцию завершает ";".

Расскажите об синтаксисе arrow function?

                    
let foo = (a, b) => a+b;

let foo2 = () => "it is string foo2";

let foo3 = (x) => {
    let y = x * 2;
    return y;
};

Это более простой и удобный синтаксис. Результат выражения справа всегда возвращается. Если аргументы передавать не нужно, то скобки оставляем пустыми. Если тело функции будет многострочным, то необходимо заключить его в фигурные скобки и обязательно добавить return.
